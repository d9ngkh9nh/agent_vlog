## Naming Convention

- `hero.jpg` — always the main article hero image
- `shop-{name}.jpg` — shop profile images (article 2)
- `species-{name}.jpg` / descriptive names — section-specific images
- Dimensions: heroes at `1400×680`, shop cards at `800×340`, portraits at `400×530`, pairs at `380×260`
